<h3>Setup & installation guide</h3>

```bash

git clone https://github.com/itsmenoahpoli/jfca-attendance-dashboard.git

cd jfca-attendance-dashboard

npm install

npm run dev

```

After, please open a web browser and go to url: `http://localhost:5173`
